## Installation 
### windows 
* Install Ruby : https://rubyinstaller.org/downloads/ 
* Install sass command : 
```java
gem install sass
```

## Hello world
* create main.scss 
```css
$primary-color: red;
.contoh{
	color : $primary-color;
}
```
* check command 
```java
sass --watch F:\data\Repository_2\ex-sass\app\main.scss:main.css
```
so, it will automatically created main.css file
